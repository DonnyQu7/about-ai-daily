## 对本项目的取舍

第一版不直接 fork 大项目，原因是目标更窄：只筛选“AI 应用落地”。因此先实现一个小而清晰的流水线：

1. 借鉴 Horizon 的采集和日报结构。
2. 借鉴 TrendRadar 的配置化和推送思路。
3. 借鉴 ai-news-radar 的 GitHub Actions 自动化方式。
4. 借鉴 github-trend 的 GitHub 项目观察方向。
5. X/Twitter 部分暂不默认实现，保留为后续可插拔 collector。
